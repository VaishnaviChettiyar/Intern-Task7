# Intern-Task7
Fetch User Data

This is a simple web project that demonstrates how to use the **JavaScript Fetch API** to retrieve and display user data from a public API.

Features

- Fetches user data from a public API
- Displays each user's name, email, and address
- Handles and displays errors when there's no internet or network issue
- Includes a Reload button to refetch the data
- Styled using basic CSS

 API Used
- URL: https://jsonplaceholder.typicode.com/users
- Returns an array of fake users with
      name
      email
      address.street, address.city
